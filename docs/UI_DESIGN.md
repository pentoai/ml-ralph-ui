# UI Design

Wireframes and interaction design for ml-ralph TUI.

## Overview

ml-ralph has two primary modes:

1. **Planning Mode** - Create PRD, view knowledge (learnings/research/stories)
2. **Monitor Mode** - Watch agent execution, view metrics

Users switch between modes via tabs. The status bar is persistent across both modes.
